# **App Name**: AuraChat

## Core Features:

- Chat Interface: A user-friendly interface with an input field for prompts and a display area for conversational exchange.
- AI Response Generation: Utilizes a large language model (LLM) as a tool to process user prompts and generate coherent, contextually relevant text responses.
- Conversation History Display: Displays the chronological flow of user queries and AI responses within the current chat session.
- New Chat Session: A simple mechanism to clear the current conversation and start a fresh chat session with the AI.

## Style Guidelines:

- Color scheme is light, evoking intelligence and modernity. The primary color, for key interactive elements, is a deep, trustworthy blue: '#285396'.
- The background color is a very light, desaturated blue-grey, providing a clean canvas: '#ECF2F7'.
- An accent color, a clear aqua, highlights important actions and information for good contrast: '#4CBCCC'.
- The primary typeface for both headlines and body text is 'Inter' (sans-serif), chosen for its excellent readability, modern aesthetic, and neutral, objective feel, perfectly suited for conveying information clearly in a chat application.
- Icons should be clean, modern, and line-based, consistent with a digital interface. Focus on intuitive symbols for chat actions like sending messages, clearing history, and general navigation.
- The layout is clean, responsive, and conversational-focused, with messages clearly separated by sender. It features a centered chat window to maintain focus on the dialogue.
- Subtle, smooth animations, such as a gentle fade-in or slide-up effect, will be used when new messages appear in the chat, enhancing the fluidity of the conversation.
